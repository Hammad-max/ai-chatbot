# 🤖 AI Customer Support Chatbot

## 📌 Description
An AI-based chatbot built using Python, Machine Learning, and NLP to handle customer queries for an e-commerce platform.

## 🚀 Features
- Intent recognition using Machine Learning
- NLP using CountVectorizer
- Automated responses
- Handles queries like order tracking, refunds, greetings

## 🛠️ Technologies
- Python
- Scikit-learn
- NLP (Natural Language Processing)
